### 🎯 搜索功能详解

#### **AND 逻辑** (默认)
```
搜索: "book AND pdf"
结果: 标题、描述、标签中**同时包含** "book" 和 "pdf" 的链接
```

#### **OR 逻辑**
```
搜索: "movie OR film"
结果: 标题、描述、标签中**至少包含** "movie" 或 "film" 的链接
```

#### **单一查询**
```
搜索: "anime"
结果: 标题、描述、标签中包含 "anime" 的链接
```
